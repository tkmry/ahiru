# 1. ゲームの目的

アヒル君はいつも不思議な宝箱を探して過ごしています。
彼は今日も旅に出て宝箱巡り。
宝箱を開けると遠くの場所へひとっ飛び！！

どの宝箱を開ければ家に帰れるのかな？


アヒル君を操作してゲームをクリアしよう。


# 2. ゲームの操作方法

## ゲームの起動

同梱している Game.exe をダブルクリックしてゲームを起動します。

## 操作方法

アヒル君の操作は簡単。

キーボードの矢印、それと Ctrl（コントロール） キーだけ。

←（左）: 左に移動
→（右）: 右に移動
↑（上）: ジャンプ(空中でもう一度ジャンプできます)
↓（下）: しゃがむ（特に意味はない）
Ctrl: 押しながら左右でダッシュ

Esc: ゲームを終了


# 3. マップエディター

同梱しているプログラムを使うと、独自のマップを作って遊べます。

## マップを作る

ステージ名を入力して「OK」をクリックして始めます。


## ステージをカスタマイズ

### 音楽

ステージで流れる曲を変更できます。
music フォルダ内にあるファイルのファイル名を入力してください。


### 最初の位置

（作った当時の事は覚えていませんが不具合があるため使用しない方がよいようです）

原文
>　とても説明しにくいです。
>　0ならば一番左から始まります。
>　640ならば一画面右から始まります。
>…予定でしたが、この機能は使わないでください、バグの元です。


### 最大の横幅

ステージの横幅です。

640 が一画面の横幅になります。


### 時間制限

秒を整数で入力してください。


## 配置するキャラの変更

メニューの[設定]→[グループ]で選択するか、画面左側のアイコンをクリックしてください。

アヒル:
    プレイヤーが使う主人公です。

敵:
    プレイヤーに襲いかかる敵キャラクターです。
    右クリックによって
     * スライム
     * スライム(赤)
     * ひよこ
     * ひよこ(灰色)
    を切り替えられます。

牛乳:
    楽してポイントが得られます。
    右クリックで
     * 牛乳
     * 牛乳2
    を切り替えられます。

宝箱:
    ワープゾーン宝箱です。
    拡張子を除いたステージ名を指定します。

    NULL という文字列の入力でゲームクリア用の宝箱を設置します。

ブロック:
    地面です。
    自キャラ、敵キャラの足場になります。
    右クリックで
     * 地面
     * 芝生付き地面
    を切り替えられます。

消しゴム:
    一度設置したオブジェクトをクリックして削除します。
